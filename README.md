# Ghosts of Tabor Skin Gallery

A static gallery site for browsing skin packs, previews, compatible items, and skinning system reference images.

## Local files

- `index.html` contains the site markup
- `styles.css` contains the site styling
- `script.js` contains the gallery behavior
- `data/skins-data.js` contains the generated gallery data
- `build-data.ps1` rebuilds the skin data files

## Publishing

This site is ready to be hosted with GitHub Pages from the `main` branch root.
